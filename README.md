# MaxxHub

MaxxHub is an Expo React Native app with a realm-themed UI, quest data, and tab-based navigation.

## Prerequisites

- Node.js 18+
- npm
- Expo Go app (for mobile testing) or iOS Simulator / Android Emulator

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the Expo development server:

```bash
npm run start
```

## Run on a Platform

- Android:

```bash
npm run android
```

- iOS:

```bash
npm run ios
```

- Web:

```bash
npm run web
```

## Tech Stack

- Expo
- React Native
- React Navigation
- Firebase
